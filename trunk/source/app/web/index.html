Ivy Tools

These Ivy tools are compatible with both Ivy 1.x and 2.x. They're all Java WebStart applications. To install them, click on one of the links below:

Ivy Browser: Search an Ivy repository for libraries, packages, and versions by key words:
Ivy Builder: Add items to an Ivy repository - prepare the directory, add the jar files, input the dependencies, generate the XML, and generate the checksums.
Ivy Grapher: Graph your Ivy reports (in the .ivy folder) to see what's not being used. Redundant or evicted revisions called out by your ivy.xml file show up in red. Click here for example images
Note that this application needs AT&T's GraphViz viewer to be installed to work (OS X and Windows) - http://www.graphviz.org/
Ivy Formatter: Format your Ivy (or XML) files, so they're nice and tidy.
