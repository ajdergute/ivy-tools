Ivy Tools

To install this:

  • Click on the link below:
    • Launch the Ivy Repository Browser application
    • Launch the Ivy Repository Builder application
    • Launch the Ivy Formatter application to beautify your Ivy files
      • Note that this program requires ATT's GraphViz to be installed to generate the diagrams - you can find that here
  • Accept all dialogs.
  • If it asks to load a newer version of Java or WebStart, say "yes".
  • Ignore the warnings about installing software on your system. I'm using the free version of Sun's WebStart architecture, and the cost of that is annoying dialogs like this for the users.
  • Launch it twice from this web page and it'll install icons on to your desktop. You won't have to visit this site again to run the appliation after that.
  • You can bookmark the link and not have to go to this web page.
  • You'll always have the most up-to-date version - you'll never have to reinstall or upgrade. Java WebStart will auto upgrade your software automatically.
  • Use the latest version of Java - Java 6 Update 11 at http://www.java.com/en/download/index.jsp
